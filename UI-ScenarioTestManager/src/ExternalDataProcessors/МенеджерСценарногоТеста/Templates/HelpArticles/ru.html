Полезные статьи, инструкции и видеоуроки

Инструменты

- Фреймворк "Тестирование 3.0" (https://github.com/ivanov660/TestingTool-3) - один из инструментов тестирования содержит полный набор для управления, создания, просмотра отчетов тестов;
- Фреймворк "Мониторинг производительности" (https://github.com/Polyplastic/1c-parsing-tech-log)- инструмент позволяет выполнять мониторинг нагрузки системы, ежедневного мониторинга, подходит для оценки нагрузочного тестирования.

- Тестирование: пример из семи шагов создания Unit-теста для платформы 1С (https://infostart.ru/public/663808/) - в статье описывается пример создания юнит теста.
