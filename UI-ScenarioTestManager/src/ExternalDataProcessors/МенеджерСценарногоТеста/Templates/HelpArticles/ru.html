Полезные статьи, инструкции и видеоуроки

Инструменты

- Фреймворк "Тестирование 3.0" (https://github.com/ivanov660/TestingTool-3) - один из инструментов тестирования содержит полный набор для управления, создания, просмотра отчетов тестов;
- Фреймворк "Мониторинг производительности" (https://github.com/Polyplastic/1c-parsing-tech-log)- инструмент позволяет выполнять мониторинг нагрузки системы, ежедневного мониторинга, подходит для оценки нагрузочного тестирования.

Статьи

- Тестирование: пример создания сценарного UI теста для платформы 1С (https://infostart.ru/1c/articles/813062/) -в статье описывается сквозной пример содания сценарного теста для бизнес процесса продажи.
- Тестирование: пример из семи шагов создания Unit-теста для платформы 1С (https://infostart.ru/public/663808/) - в статье описывается пример создания юнит теста.
- Тестирование: просмотр результатов тестов в 1С Предприятии Allure Skin (https://infostart.ru/public/821772/) - в статье описывается пример как увидеть результаты тестов в предприятии 1С.
- Тестирование: запуск и контроль выполнения автоматизированнных тестов в 1С Предприятии (https://infostart.ru/public/856724/) - в статье описывается использование конфигурации 1С в качестве системы CI для запуска автоматизированных тестов.
- Тестирование - игровое моделирование (https://infostart.ru/public/1644998/) - в статье описывается подход к созданию автоматизированных тестов.
